

# Weather Report
Get todays forecast.

## Getting Started
 ### Prerequisites
 * Java 8
 * Gradle 5

### Running
 To run the program:

``` /gradlew bootRun ```

Go to the url:
```  http://localhost:8080/weatherRequest ```
